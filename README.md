# awesome_gfx
A collection of open source projects for low level graphics dev

## Texture Baking

- [Fornos](https://github.com/caosdoar/Fornos) GPU Texture Baking Tool. A fast and simple tool to bake your high-poly mesh details to textures.
- [AwesomeBump](https://github.com/kmkolasinski/AwesomeBump) AwesomeBump is a free program written using Qt library designed to generate normal, height, specular or ambient occlusion textures from a single image. Since the image processing is done in 99% on GPU the program runs very fast and all the parameters can be changed in real time.
